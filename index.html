Sobre

Somos um grupo internacional, com foco em ajudar mais mulheres a tornarem-se participantes ativas e líderes de comunidades Python de código aberto. Nossa missão é promover, educar e impulsionar a existência de uma comunidade Python diversificada através de sensibilidação, aducação, conferências, eventos e encontros sociais.
O PyLadies também tem como objetivo fornecer uma rede de apoio amigável para as mulheres e uma ponte para o Mundo Python. Qualquer pessoa com um interesse em Python é encorajada a participar - contato que seja mulher.
